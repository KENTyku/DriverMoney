Задание:
есть два вида водителей; 
с почасовой оплатой
фиксированной оплатой.

Приветствуется поддержка стратегии оплаты в качестве входного параметра.

Для каждого вида необходимо рассчитывать заработную плату. Для повременщиков» формула для расчета такова: 
«среднемесячная заработная плата = 20.8 * 8 * почасовую ставку», 
для работников с фиксированной оплатой: «среднемесячная заработная плата = фиксированной месячной оплате».

Необходимо разработать консольное приложение, позволяющее выполнять следующие операции со списком водителей:	
загружать список водителей из файла
сохранять список водителей в файл
*добавлять, удалять водителя в список
сортировать водителей по убыванию/возрастанию среднемесячного заработка (при совпадении зарплаты - 
    упорядочивать данные по ФИО в лексикографическом порядке)
выводить на экран ФИО и среднемесячный заработок для всех элементов списка
выводить на экран N первых элемента из списка

Приветствуется проверка входного файла на ошибки
